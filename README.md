# DND Todo List

## Information
This is a drag and drop task manager
  ##### Tech Used:
- React
- Tailwind CSS
- Typescript
- Vite

#### Setup
1. Install Vite to project <br>
`npm create vite@latest .`<br>
-> framework: React<br>
-> variant: TypeScript
2. Install Tailwind CSS
```
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```
